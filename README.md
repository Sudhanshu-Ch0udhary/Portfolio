# Portfolio  

A minimal and responsive personal portfolio built with **SolidJS**, styled using **Tailwind CSS**, and enhanced with smooth animations via **Motion One**.

Designed to be fast, accessible, and focused on showcasing projects and skills without unnecessary complexity.

---

## Features

- Built with **SolidJS** for fine-grained reactivity and performance
- Styled using **Tailwind CSS**
- Subtle animations using **Motion One**
- Fully responsive layout
- Clean, minimal UI
- Optimized for fast load times

---

## Tech Stack

- **Framework:** SolidJS  
- **Styling:** Tailwind CSS  
- **Animations:** Motion One  
- **Build Tooling:** Vite  
- **Deployment:** Vercel

---

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Sudhanshu-Ch0udhary/Portfolio.git
cd portfolio
npm install 
npm run dev 
```
